

# Developer Portfolio Website

A modern, polished static portfolio website with light/dark theme support and smooth animations.

## Pages & Sections (Single Page Application)

### 1. Navigation Bar
- Text-based logo with developer name
- Navigation links: Profile, Projects, Skills, Contact
- Smooth scroll to each section
- Light/Dark theme toggle with sun/moon icon
- Sticky top positioning with subtle backdrop blur

### 2. Hero / Profile Section
- Profile image (placeholder avatar)
- Name and "Full Stack Developer" title
- Short professional introduction paragraph
- Two CTA buttons: "View Projects" and "Contact Me"
- Clean centered layout with subtle entrance animation

### 3. About Section
- Professional summary paragraph
- Key strengths displayed as highlighted cards (e.g., Problem Solver, Team Player, Fast Learner)

### 4. Projects Section
- Responsive grid of project cards (3 columns desktop, 2 tablet, 1 mobile)
- Each card: image, title, description, tech stack badges, GitHub & Live Demo buttons
- Hover effects: subtle lift/shadow animation
- Sample placeholder projects included

### 5. Skills Section
- Skills organized into 4 categories: Frontend, Backend, Database, DevOps
- Clean badge/chip design for each skill
- Lucide line-style icons for each category header

## Design & Theme
- **Primary color:** #2962FF (blue)
- **Accent color:** #7DF9FF (electric cyan)
- **Light mode background:** #E0E5EC (soft gray)
- **Dark mode background:** #333333
- **Font:** Inter throughout
- CSS variables for all theme colors
- Smooth 300ms theme transitions
- Theme preference saved to localStorage
- Fully responsive across mobile, tablet, and desktop
- Clean spacing, modern minimal aesthetic, production-ready polish
